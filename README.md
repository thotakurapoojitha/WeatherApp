# WeatherApp
🌊 WeatherWave ⛅

A quiet stretch of sky finds its way into your screen. Tiny clouds drift through the frame you’re staring at. This moment stays still, just long enough to notice. ☁️✨

Floating shapes drift across the screen when you open a new page. Soft colors blend like morning light spreading through clouds 🌅. Little details hover in place — your schedule, temperature, maybe a note to yourself 📝. This is how WeatherWave fills the space around you.

Starts with nothing prebuilt. Skip the setup steps ⚡. It runs on basic web tools only — HTML, CSS, then JavaScript stitched into a solid free weather API that does more than expected.

🌱 Why this exists

Most weather apps are one of two types:

📊 Too many things on a screen at once
📄 Plain text dumps

A calm screen shows just what matters at this moment 🧘‍♀️. Temperature sits quietly alongside how it truly feels outside 🌡️. Humidity moves slowly through the display 💧, followed by wind speed gliding in 🌬️. Pressure reads steady near the edge 📈.

Nothing interrupts. No noise fills the space.

This is WeatherWave — still, clear, speaking only when needed.

🚀 Here is what happens when you use it
📍 Follow you around

Starts by asking if it can check where you are through the browser 🌍. Once allowed, pulls up the local weather automatically. No extra steps needed.

🔍 Jump anywhere

Start typing a place — like Hyderabad, Tokyo, or London — and right away the present conditions appear 🌆.

🌡️ Temperature your way

Click the temperature label once — switch from Celsius to Fahrenheit instantly 🔁.
One tap. No friction.

🌤️ Glanceable forecast

A week’s worth of weather predictions shows up right away 📅.

Default values appear instantly ⚡

Real data replaces them smoothly 🔄

Layout stays clean and stable 🧩

Blocks line up side by side, shifting gently as new forecasts arrive. Ready-built slots wait quietly for real values.

🎨 Feel good to look at

Floating glass-style panels 🧊

Soft light gradients 🌈

Subtle animated particles ✨

Everything is designed to feel calm, not crowded.

⚙️ Key Implementation Details

Uses Fetch API for asynchronous data handling

Implements async/await for clean promise management

Dynamically updates DOM with real-time data

Handles invalid city inputs gracefully

Displays:

🌡 Current temperature

🌬 Wind speed

📅 7-day forecast

🌤 Weather conditions
